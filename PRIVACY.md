# Privacy Policy

Discord Image Blocker does not collect, store, sell, share, or transmit user data.

## Data Collection

The extension does not collect any of the following:

- Personally identifiable information
- Health information
- Financial or payment information
- Authentication information
- Personal communications
- Location data
- Web history
- User activity
- Website content

## How The Extension Works

Discord Image Blocker runs a local content script on `discord.com`. The script looks for image-related media elements in the current Discord page and hides them in the browser.

This processing happens locally on the user's device. The extension does not send page content, Discord messages, browsing activity, or any other information to a server.

## Remote Code

The extension does not use remote code. All JavaScript is included in the extension package.

## Analytics And Tracking

The extension does not include analytics, tracking, telemetry, advertising scripts, or external network requests.

## Third Parties

The extension does not sell, transfer, or disclose user data to third parties.
